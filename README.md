## Projects

### 🏠 [chat-property-whisperer](https://github.com/Lucastirbat/chat-property-whisperer)
> *Rental search, reimagined as a conversation.*

Skip the filter forms. Tell the AI what you want — neighborhood vibe, budget, must-have amenities — and it intelligently searches Zillow, Apartments.com, Realtor.com, and ApartmentList simultaneously. Built on Claude AI with a custom MCP integration via Apify, it deduplicates results across platforms and surfaces the best matches in a unified feed.

`React 18` `TypeScript` `Node/Express` `Claude AI` `Apify MCP` `SSE` `Tailwind` `Shadcn UI`
